Life history evolution in a community context.

You must first create a folder in the same directory as the code and title it "Figures". This allows the export of the figures to this folder.

Then you can use the R File "1 NicheShfitandLHAge" to run the analyses for the age-dependent model. 

The size-dependent model begins with the R file "8 NicheShfitandLHSize".

Both sets of code are annotated to help you understand what each step is doing.
